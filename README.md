# SHS Tube

YouTube app for Android with background playback, Picture-in-Picture, and downloads.

**Developer:** SHS  
**Telegram:** https://t.me/aamoviesofficial

## Features
- Background audio playback (screen off)
- Picture-in-Picture mode
- Video & audio downloads
- Media notification controls (play/pause/next/prev)
- No ads

## Download

Get the latest APKs from [Releases](https://github.com/hamsazzad/SHS-Tube/releases).

| APK | Description |
|-----|-------------|
| SHS-Tube-v1.0-arm64-v8a.apk | 64-bit ARM (most modern phones) |
| SHS-Tube-v1.0-armeabi-v7a.apk | 32-bit ARM (older phones) |
| SHS-Tube-v1.0-x86_64.apk | 64-bit x86 (Intel/AMD) |
| SHS-Tube-v1.0-x86.apk | 32-bit x86 |
| SHS-Tube-v1.0-universal.apk | All devices |
| SHS-Tube-v1.0-Clone-*.apk | Install alongside standard version |

## Build from Source
```
git clone https://github.com/hamsazzad/SHS-Tube.git
cd SHS-Tube
./gradlew assembleDebug
```
